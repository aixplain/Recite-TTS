# Recite TTS
الرخصة: MIT
عنوان URI للرخصة: https://choosealicense.com/licenses/mit/


## الوصف
أضف مقاطع صوتية منطوقة إلى أي مدونة باستخدام API واحد. تستفيد هذه الإضافة مفتوحة المصدر لـ WordPress من مصمم خط المعالجة في aiXplain لتحويل منشورات المدونة رقميًا إلى أي لغة قابلة للتخصيص من قِبَل المستخدم، مع إمكانية اختيار الصوت. العضوية مجانية، وتتضمن رصيدًا إضافيًا عند التسجيل لاستكشاف جميع خيارات المنصة.


## التثبيت
1. ارفع/استنسخ المستودع إلى مجلد `/wp-content/plugins/`
2. فعِّل الإضافة من قائمة 'Plugins' في WordPress
3. سجِّل واعثر على الصوت الذي يناسبك (ابحث عن نماذج التعرف على الكلام) في [aiXplain.com](https://aixplain.com)
4. اشترك واحصل على عنوان URL ومفتاح API للنموذج المختار، مع مراعاة أن بعض النماذج لديها قيود على عدد الأحرف.
4. انتقل إلى صفحة إعدادات الإضافة في لوحة إدارة WordPress
5. حدِّث `api key` و`api url`
6. يمكنك تشغيل مهمة في الخلفية لمعالجة جميع المنشورات بالنقر على `process X Post(s)` أو تحديد المنشورات التي تريد معالجتها بشكل فردي
